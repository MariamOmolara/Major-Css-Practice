CSS stands for Cascading Style sheet and is used for the styling and design of a website. It is one of the many languages a web programmer uses to create a website. There are 3 types of CSS;

  • In line CSS
  • Internal CSS
  • External CSS

There are 5 CSS selectors

  • CSS Universal Selector
  • CSS Element selector
  • CSS ID selector
  • CSS Class Selector
  • CSS Attribute Selector

Difference between paddings, margins, and borders.

Padding:

CSS Padding is used to create space around an element's content, inside of any defined borders.

Margins:

CSS Margins are used to create space around elements, outside of any defined borders.

Border:

CSS border properties allow you to specify the style, width, and color of an element's border.
